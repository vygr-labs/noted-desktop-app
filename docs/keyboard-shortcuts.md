# Keyboard Shortcuts

## Global

| Shortcut | Action |
|----------|--------|
| `Ctrl+Shift+F` | Open command palette (global search) |
| `Ctrl+F` | Find in current note |
| `Ctrl+H` | Find & replace in current note |
| `Ctrl+[` | Toggle note list sidebar |
| `Ctrl+Alt+N` | Quick capture |

## Editor (Rich Text)

| Shortcut | Action |
|----------|--------|
| `Ctrl+B` | Bold |
| `Ctrl+I` | Italic |
| `Ctrl+U` | Underline |
| `Ctrl+Shift+S` | Strikethrough |
| `Ctrl+Shift+B` | Blockquote |
| `Ctrl+Alt+C` | Code block |

## In-Note Search

| Shortcut | Action |
|----------|--------|
| `Enter` | Next match |
| `Shift+Enter` | Previous match |
| `Escape` | Close search |

## Replace (when visible)

| Shortcut | Action |
|----------|--------|
| `Enter` | Replace current match |
| `Escape` | Close search & replace |
